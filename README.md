# homebrew-outpost

Private Homebrew tap hosting Dragonfly's operator-fleet formulae.

## Use

```bash
brew tap dragonflyic/outpost https://github.com/dragonflyic/homebrew-outpost.git
brew install outpost-supervisor
```

## Formulae

- `outpost-supervisor` — operator-side brew reconciler ([source](https://github.com/dragonflyic/outpost))
